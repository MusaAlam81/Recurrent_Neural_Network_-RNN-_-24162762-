# Recurrent Neural Networks: Teaching a Machine to Remember

A machine learning tutorial demonstrating how RNNs learn from sequential data, using the Jena Climate dataset to predict daily temperatures. The key experiment explores how window size (how much history the RNN sees) affects prediction accuracy.

# Repository Structure

index.html        # Tutorial web page (<2000 words)
notebook.ipynb    # Full Jupyter notebook with all code & figures
figures/          # Plots generated by the notebook (auto-created on run)
README.md         # This file
LICENSE           # MIT Licence

# Getting Started
# 1. Clone the repository
bash
git clone https://github.com/YOUR_USERNAME/rnn-tutorial.git
cd rnn-tutorial

# 2. Install dependencies
bash
pip install tensorflow numpy pandas matplotlib scikit-learn jupyter

# 3. Run the notebook
bash
jupyter notebook notebook.ipynb
Click Kernel → Restart & Run All. The dataset downloads automatically. All 5 figures are saved to `figures/`.

# 4. View the tutorial
Open index.html in your browser. Replace figure placeholders with the generated images.

# What's Covered

Why standard MLPs fail on sequential data
The RNN hidden state — how memory works
Backpropagation Through Time (BPTT)
The sliding window technique for time series
Building and training an RNN in Keras
Key experiment: How window size affects prediction accuracy
Limitations of vanilla RNNs and why LSTMs exist

# Dataset

Jena Climate Dataset: Hourly weather readings from Jena, Germany (2009 to 2016), resampled to daily mean temperature.

Source: Max Planck Institute for Biogeochemistry. Available via [TensorFlow Datasets](https://www.tensorflow.org/tutorials/structured_data/time_series).
